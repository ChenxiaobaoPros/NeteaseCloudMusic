# NeteaseCloudMusic
网易云音乐的WPF实现，基于prism mvvm模式
由于时间仓促，部分功能和异常尚未完成，web api接口地址(http://47.101.43.134/api )
# 参考
api部分参考 [Binaryify](https://github.com/Binaryify/NeteaseCloudMusicApi)的 node.js代码
